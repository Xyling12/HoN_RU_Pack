# Перевод .str файла

// turbo-all

## Правила перевода

1. **Игровые термины** — транслитерировать на русский (например, "All Pick" → "Олл Пик", "Mid Wars" оставить как есть)
2. **Имена собственные** — оставлять имена героев, предметов и способностей на английском, если нет устоявшейся русской транслитерации
3. **Названия клавиш** — оставлять на английском (Ctrl, Shift, Alt и т.д.)
4. **Цветовые коды** — сохранять `^xxx` и `^*` в точности как есть
5. **Плейсхолдеры** — сохранять `{переменная}` в точности как есть
6. **Форматирование** — сохранять `\\n` переносы строк, табуляцию между ключом и значением
7. **Комментарии** — оставлять строки с `//` как есть
8. **Тон** — естественный, понятный русский; избегать излишне формальных или машинных формулировок
9. **Бан/разбан** — использовать «забанен/разбанен» (не «бабанин/разбоен»)
10. **Согласованность** — проверять существующие переводы в уже переведённых файлах для единообразия терминов

## Шаги

1. **Оценить файл**: Открыть целевой `.str` файл и подсчитать общее количество строк. Определить, какие строки уже переведены, а какие ещё на английском.

2. **Создать пакетный скрипт** в `c:\tmp\` (например, `fix_tr_<имяфайла>.py`), который выполняет поиск и замену для простых переводов. Использовать Python с regex для сохранения форматирования. Обрабатывать партиями по ~1500 строк.

3. **Запустить пакетный скрипт** и проверить изменения через `git diff --stat`.

4. **Ручная проверка**: Просмотреть каждую партию визуально, исправить неуклюжие формулировки, артефакты машинного перевода или несоответствия.

5. **Grep-верификация**: Поиск часто встречающихся непереведённых английских шаблонов, чтобы убедиться, что ничего не пропущено:
   ```
   grep -P "^\S+\t+[A-Z][a-z]" <файл>  # строки начинающиеся с ключа + английский текст
   ```

6. **Обновить task.md** с прогрессом после каждой партии.

7. **Скопировать в dist**: После завершения всех переводов скопировать обновлённый файл в `dist\installer_payload_stage\bundle\`.

## Верификация

- Запустить `grep -c "^\S" <файл>` для подсчёта общего количества ключей
- Поиск оставшихся строк только на английском
- Убедиться, что цветовые коды и плейсхолдеры не повреждены
- Сравнить количество ключей до и после, чтобы убедиться, что ни один ключ не потерян
