# HoN RU Pack - обход блокировок

## Что это

Это русский языковой пакет для Heroes of Newerth с опциональным обходом блокировок. Обход нужен в тех случаях, когда ограничения провайдера или DPI мешают подключению к игре и связанным сервисам.

## Как это работает

Установщик может дополнительно настроить `AmneziaWG` - клиент на базе WireGuard с обфускацией трафика. Через туннель идет только трафик к выбранным сервисам, а остальной интернет продолжает работать напрямую.

Это не "полный VPN на всю систему", а раздельная маршрутизация только для нужных направлений.

## Какие сервисы можно направить через обход

| Сервис | Назначение |
| --- | --- |
| `HoN` | Серверы Heroes of Newerth, основной сценарий использования |
| `YouTube` | Видео и CDN Google |
| `Discord` | Голосовая связь и мессенджер |
| `Telegram` | Мессенджер |
| `ChatGPT / OpenAI` | Сервисы OpenAI |

Пользователь сам выбирает, что именно направлять через обход.

## Почему нужны права администратора

Установщик запрашивает права администратора, потому что:

- установка клиента `AmneziaWG` требует доступа к системным каталогам;
- создание сетевого туннеля выполняется на уровне системы;
- создается задача автозапуска для служебного агента.

Это стандартное требование для такого типа сетевого клиента.

## Что устанавливается

1. Русские файлы локализации для HoN.
2. `AmneziaWG`, если выбран обход блокировок.
3. Конфигурация туннеля и данные регистрации клиента.
4. Автозапуск служебного агента HoN RU Pack.

## Безопасность

Что важно понимать:

- используется проект `AmneziaWG` с открытым исходным кодом;
- через туннель направляется только выбранный трафик;
- установщик не добавляет рекламу и постороннее ПО;
- для работы обхода используются сетевые параметры и конфигурация туннеля, необходимые для подключения.

## Как удалить

Запустите `uninstall_hon_ru_pack.ps1`. Скрипт удалит:

- русские файлы локализации;
- настроенный туннель `AmneziaWG`;
- связанные служебные файлы и автозапуск.

При необходимости `AmneziaWG` можно удалить и вручную через параметры Windows.

## Технические детали

- Протокол: `AmneziaWG`
- Транспорт: `UDP`
- Конфигурация: раздельная маршрутизация по списку IP-адресов
- Назначение: обход сетевых ограничений для выбранных сервисов
