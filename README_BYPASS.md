# HoN RU Pack — Обход блокировки

## Что это?

Русский языковой пакет для Heroes of Newerth с **опциональным обходом блокировок**,
которые мешают играть из России (РКН заблокировал серверы игры).

## Как работает обход?

Инсталлер устанавливает **AmneziaWG** — это модифицированная версия WireGuard (открытый VPN-протокол).
Через VPN-тоннель проходит **ТОЛЬКО трафик к заблокированным сервисам** — весь остальной
интернет работает напрямую, без VPN, без потери скорости.

### Какие сервисы можно разблокировать:

| Сервис | Описание |
|--------|----------|
| ⚡ **HoN** | Серверы Heroes of Newerth (рекомендуется ✅) |
| 📺 **YouTube** | Видеохостинг Google |
| 💬 **Discord** | Мессенджер для голосовой связи |
| ✈ **Telegram** | Мессенджер (на случай очередных блокировок) |
| 🧠 **ChatGPT** | OpenAI / ChatGPT |

Вы сами выбираете что разблокировать — ничего лишнего не включается.

## Почему нужны права администратора?

Инсталлер запрашивает права администратора (щиток UAC) потому что:
- Установка VPN-клиента AmneziaWG требует доступа к системным папкам
- Создание VPN-тоннеля — это системная сетевая операция
- Запланированная задача для автообновления русского пакета

**Это стандартная процедура для любого VPN-клиента.** Никакие пароли, данные,
или личная информация не собираются и никуда не отправляются.

## Что именно устанавливается?

1. **Русские строки** — файлы перевода копируются в папку игры
2. **AmneziaWG** (если выбрали обход) — VPN-клиент, работает как фоновый сервис Windows
3. **Автообновление** — запланированная задача, которая следит за актуальностью перевода

## Безопасно ли это?

✅ **Да.** Вот почему:

- **Открытый код**: AmneziaWG — [открытый проект на GitHub](https://github.com/amnezia-vpn/amneziawg-windows-client)
- **Split-tunnel**: через VPN идёт ТОЛЬКО трафик к выбранным сервисам, остальное — напрямую
- **Нет логирования**: VPN-сервер не ведёт логов и не хранит историю посещений
- **Нет рекламы**: инсталлер ничего не рекламирует и не устанавливает лишнего
- **Без трекинга**: никакая аналитика или телеметрия не собирается

## Как удалить?

Запустите `uninstall_hon_ru_pack.ps1` — он полностью удалит:
- AmneziaWG и VPN-тоннель
- Русские файлы перевода
- Запланированную задачу автообновления

Или вручную: Параметры → Приложения → AmneziaWG → Удалить.

## Технические детали

- **VPN-протокол**: AmneziaWG (WireGuard с обфускацией трафика)
- **Сервер**: Нидерланды, выделенный VPS
- **Порт**: 51820/UDP
- **Шифрование**: Curve25519 + ChaCha20-Poly1305 (стандартное для WireGuard)
- **Обфускация**: Junk-пакеты для обхода DPI (Deep Packet Inspection)
