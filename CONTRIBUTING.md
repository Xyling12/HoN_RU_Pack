# 🎮 HoN RU Pack — Гид контрибьютора

Добро пожаловать! Этот документ объясняет как устроен проект, как работать с переводом и как собрать инсталлятор.

## Структура проекта

```
HoN_RU_Pack/
├── bundle/                  ← Все .str файлы (локализация)
│   ├── entities_en.str      ← Главный файл: умения, предметы, герои (125k строк!)
│   ├── interface_en.str     ← Интерфейс, меню, настройки
│   ├── client_messages_en.str
│   ├── game_messages_en.str
│   └── bot_messages_en.str
├── docs/                    ← Документация (ты здесь)
├── assets/                  ← Иконки для exe
├── dist/                    ← Собранные exe (не в git)
├── sanitize_bundle.py       ← Обязательно запускать перед установкой
├── install_hon_ru_pack.ps1  ← Установка локально
├── build_hon_ru_installer_exe.ps1   ← Сборка инсталлятора
├── build_hon_ru_uninstaller_exe.ps1 ← Сборка деинсталлятора
└── version.txt              ← Текущая версия
```

## Быстрый старт для контрибьютора

1. Клонируй репозиторий
2. Открой нужный `.str` файл в `bundle/`
3. Отредактируй строки (формат: `КЛЮЧ\tЗНАЧЕНИЕ`)
4. Запусти `python sanitize_bundle.py`
5. Запусти `.\install_hon_ru_pack.ps1` для проверки в игре
6. Сделай коммит
