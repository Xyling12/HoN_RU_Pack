# 🎮 HoN RU Pack v1.5 — Патч-ноут

## ⚡ Обход блокировки (Zapret)

- Полностью переработана система обхода DPI-блокировки
- Интегрирован **HoN-модифицированный Zapret** (ALT11) с поддержкой серверов Juvio
- Добавлены IP-адреса игровых серверов HoN для корректного обхода
- Добавлено специальное правило для игровых UDP-портов (10000-10100)
- Исправлен баг с потерей системных DLL при установке
- Zapret устанавливается как Windows-сервис и запускается автоматически

## 🌐 Русификация

- Имена всех **героев** оставлены на английском языке — как в оригинале
- Названия всех **предметов** оставлены на английском языке
- Кнопки лобби ("Join Legion" / "Join Hellbourne") на английском
- Названия ролей (Carry, Mid, Offlane и др.) на английском
- Описания героев, способностей, предметов и весь остальной интерфейс — **на русском**

## 📦 Два варианта установщика

| Файл | Описание |
|------|----------|
| `HoN_RU_Pack_Installer.exe` | Только русификатор |
| `HoN_RU_Pack_Installer_Bypass.exe` | Русификатор + обход блокировки для РФ |

## 📋 Как установить

1. Скачай нужный `.exe`
2. Запусти от **администратора**
3. Нажми "Установить"
4. Готово! Запускай HoN через Juvio

> **Для РФ**: используй версию с Bypass — она автоматически установит Zapret для обхода блокировки.
