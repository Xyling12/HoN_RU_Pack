# 🛒 Работа с переводом предметов (Items)

## Где находятся предметы

Все предметы — в файле `bundle/entities_en.str`.

Строки предметов начинаются с `Item_` или `Recipe_`. Например:
```
Item_AxeOfTheMalphai_name           Топор Малфаев
Item_AxeOfTheMalphai_description    Пассивно: каждый удар...
Item_AxeOfTheMalphai_flavortext     "Куй железо пока..."
```

## Как найти нужный предмет

Открой `bundle/entities_en.str` в любом текстовом редакторе (рекомендую **VS Code** или **Notepad++**).

Поиск: `Ctrl+F` → введи `Item_НазваниеПредмета`

Например:
- `Item_BloodArmor` — Кровавый доспех
- `Item_MysticVestments` — Мантия Мистика
- `Item_MarkofthePredator` — Знак Хищника

## Поля предмета

| Ключ | Что это | Приоритет |
|---|---|---|
| `Item_X_name` | Название предмета | ⭐⭐⭐ Высокий |
| `Item_X_description` / `Item_X_description_simple` | Описание в магазине | ⭐⭐⭐ Высокий |
| `Item_X_flavortext` | Лорный текст курсивом | ⭐⭐ Средний |
| `Recipe_X_name` | Название рецепта | ⭐ Если неверно |

## Пример работы

**Нашёл строку:**
```
Item_BloodborneMaul_description    При повреждении получает 1 заряд за каждые 10 потерянных здоровья...
```

**Что улучшить:**
- "При повреждении" → "При получении урона"
- "потерянных здоровья" → "потерянного здоровья"

**После:**
```
Item_BloodborneMaul_description    При получении урона набирает 1 заряд за каждые 10 потерянного здоровья...
```

## Правила для предметов

### Названия (name)
- Должны быть лаконичными: 2-4 слова max
- Соответствовать функции: щит → "Щит ...", меч → "Клинок ..."
- Если в оригинале имя собственное (Shrunken Head, Codex) — **можно оставить или перевести**

### Описания (description)
- Кратко — они отображаются в небольшом тултипе
- Пассивный эффект: начинай с "Пассивно:" или "При ..."
- Активный эффект: начинай с глагола действия ("Применяет", "Создаёт", "Наносит")
- Аура: начинай с "Аура:"

### Лор (flavortext)
- Свободный стиль, художественный перевод
- Кавычки → «русские кавычки»
- Можно немного адаптировать — главное передать атмосферу

## Сокращения в описаниях

| Длинная форма | Принятое сокращение |
|---|---|
| Скорость движения | Скор. движ. |
| Скорость атаки | Скор. атаки |
| Перезарядка | Перезарядка (не сокращать) |
| максимальное здоровье | макс. здоровье |
| магическая броня | маг. броня |
| за секунду | в сек. |
| физический урон | физ. урон |
| магический урон | маг. урон |

## Workflow: как проверить результат

1. Отредактировал строку в `bundle/entities_en.str`
2. Запусти:
   ```powershell
   python sanitize_bundle.py
   .\install_hon_ru_pack.ps1
   ```
3. Открой игру → зайди в магазин → найди предмет
4. Проверь что текст отображается корректно и не вылазит за рамки

## Типичные проблемы

| Проблема | Решение |
|---|---|
| Слишком длинный текст в тултипе | Сократи описание |
| Артефакт `^` в тексте | Это цветовой код, не трогай |
| Текст не обновился в игре | Перезапусти игру, не просто карту |
| Ошибка sanitize_bundle | Проверь что TAB (не пробел) между ключом и значением |
